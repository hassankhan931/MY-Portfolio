# 💼 Hassan Khan – Developer Portfolio

🚀 **Live Website:** [hassan-khan-portfolio.netlify.app](https://hassan-khan-portfolio.netlify.app)

Welcome to my personal portfolio! This is a showcase of my skills, projects, and passion for frontend development. Built with **React.js** and **Tailwind CSS**, this site reflects my journey as a developer and the real-world projects I've crafted.

---

## 🛠️ Tech Stack

- ⚛️ **React.js** – Component-based, dynamic UI
- 🎨 **Tailwind CSS** – Utility-first responsive design
- ⚙️ **Vite** – Lightning-fast bundler & dev server
- 🌙 **Dark Mode** – With sound & vibration toggle
- 📦 **EmailJS** – Contact form email functionality
- 🧠 **Custom Scripts** – Typewriter, animations, intersection observers

---

## 📂 Features

- ✅ Fully responsive design
- ✅ Animated typewriter intro
- ✅ Scroll-triggered section reveals
- ✅ Dark/light mode with sound & vibration
- ✅ Email contact form (via EmailJS)
- ✅ Projects section with real apps
- ✅ SEO-friendly with `react-helmet`
